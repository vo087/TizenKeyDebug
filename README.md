
For Samsung Smart TV Tizen <br>
(my TV is Tizen 8) <br>  

This is an example project of a Tizen WGT/web app. <br>
(while the functionality of it should maybe be a TizenBrew module due to simplicity) <br>  

The name is somewhat arbitrary, and may not be accurate. 

### What it does
Simple web app website that prints what input is coming in, from Remote or keyboard etc.


### Installation
I use TizenBrewInstaller (by risxd), on PC to add TizenBrewInstaller as a local TV app. <br>
In the TizenBrewInstaller TV app, I add this github repo and it installs this app for me. <br>
(Note: I mean TizenBrewInstaller, not TizenBrew)

Of course there are many ways to install, but I don't use them. (so don't ask me) <br>
